# rPets
A MiniaturePets addon made for Retrox.eu


## Placeholders
All petnames are lowercased, alpha-numeric strings

#### rpets_xp
This placeholder represents the xp value of a pet <br>
e.g: ``` %rpets_xp_boxer% ```


#### rpets_lvl
This placeholder represents the lvl of a pet <br>
e.g: ``` %rpets_xp_minime% ```
